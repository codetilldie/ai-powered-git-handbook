## 附录：资源与工具

学习 Git 的道路永无止境。除了本手册，还有许多优秀的资源可以帮助你巩固所学、提升技能，以及在遇到困难时寻求帮助。

### 官方与权威文档

- **Pro Git Book (中文版)**
  - **链接**: [https://git-scm.com/book/zh/v2](https://git-scm.com/book/zh/v2)
  - **简介**: 这是学习 Git 最权威、最全面的免费在线书籍，由 Git 官方推荐。本书从基础到高级，深入讲解了 Git 的内部原理和各种命令。本手册的许多内容也深受其启发。

- **Git 官方文档**
  - **链接**: [https://git-scm.com/doc](https://git-scm.com/doc)
  - **简介**: 最原始、最准确的 Git 命令参考手册。当你对某个命令的特定参数有疑问时，这里是最终的答案。

### 在线练习平台

- **Learn Git Branching**
  - **链接**: [https://learngitbranching.js.org/](https://learngitbranching.js.org/)
  - **简介**: 一个交互式、可视化的 Git 学习网站。它通过一系列关卡任务，让你在虚拟环境中动手操作，直观地理解分支、合并、变基等复杂概念。强烈推荐给初学者。

- **Atlassian 的 Git 教程**
  - **链接**: [https://www.atlassian.com/git/tutorials](https://www.atlassian.com/git/tutorials)
  - **简介**: Bitbucket 的母公司 Atlassian 提供的免费教程，内容清晰、组织良好，覆盖了从基础到高级工作流的方方面面。

### GUI 客户端工具推荐

虽然我们鼓励你熟练使用命令行，但一个好的图形用户界面（GUI）工具可以在特定场景下极大提升效率，例如查看复杂的提交历史、处理合并冲突等。

- **Visual Studio Code**
  - **简介**: VS Code 内置了非常强大的 Git 支持，可以直接在编辑器内完成绝大部分日常操作（暂存、提交、推送、拉取、解决冲突等）。配合 `GitLens` 和 `Git Graph` 等插件，体验更佳。

- **SourceTree**
  - **平台**: Windows, macOS
  - **简介**: 一款免费但功能强大的 Git GUI 客户端。它的提交历史可视化做得非常出色，能让你清晰地看到每一次分支和合并。

- **GitKraken**
  - **平台**: Windows, macOS, Linux
  - **简介**: 界面现代化、功能丰富的商业 Git GUI 工具（对个人和开源项目免费）。它将工作流、冲突解决和代码审查无缝集成在一起。

### 社区与支持

- **Stack Overflow**
  - **链接**: [https://stackoverflow.com/questions/tagged/git](https://stackoverflow.com/questions/tagged/git)
  - **简介**: 当你遇到任何具体的、奇怪的 Git 问题时，很可能有人已经在这里问过并得到了解答。这是寻求具体问题答案的最佳场所。

- **GitHub Community**
  - **链接**: [https://github.community/](https://github.community/)
  - **简介**: 围绕 GitHub 平台的官方社区，讨论内容包括 Git 使用、GitHub Actions、API 等。
